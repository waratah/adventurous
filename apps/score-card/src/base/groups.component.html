Overview of the system

This system allows you to produce the PDF required by Scouts NSW. If possible it allows questions to be answered once, like your first aid certificate. Just because you have completed and been verified by this system does not mean you have completed the qualification. You must submit the application to Scouts NSW Activities Training

You must complete the Adventurous Log book with all your activities. Get your copy here:

It is not meant as a pixel perfect example of the PDF. Some questions are changed to allow it to be entered into the simple data system this application uses

The bars on the sections are coloured depending on the level. These colours echo Cubs (Safe Participant), Scouts (Trained Participant), Venturers (Assistant Guide) and Rovers (Guide).

There are also region specific training which is in draft. Stay tuned on advances in the scouting approval system.
